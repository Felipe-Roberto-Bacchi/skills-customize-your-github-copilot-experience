
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Create a Hangman game where players guess letters to reveal a hidden word before running out of attempts. You'll practice string manipulation, loops, conditionals, and random selection.

## 📝 Tasks

### 🛠️ Build the Hangman Game

#### Description

Develop a fully functional Hangman game that implements the core game mechanics. The game should present players with a hidden word and allow them to guess letters until they either correctly identify the word or exhaust their attempts.

#### Requirements

Completed program should:

- Randomly select words from a predefined list
- Accept letter guesses and display current progress in underscore format (_ _ _)
- Track and display the number of incorrect guesses remaining
- End the game when the word is guessed or attempts are exhausted
- Display appropriate win/lose messages at the end of each game
